Repository of sample projects based on .NET MAUI Preview 10.

Implementation of **Rock Paper Scissors** with **[CommunityToolkit.Maui.Markup](https://github.com/CommunityToolkit/Maui.Markup)**.
Demonstrates:
- Simple navigation
- C# only UI
- MVVM
